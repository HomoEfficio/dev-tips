# vi 환경 설정

https://ourcstory.tistory.com/38 여기에 자세히 나와있고, 대략 아래 내용만 해도 쓸만함

```
syntax on                                                                        
set hlsearch
set nu
set smartindent
set tabstop=4
set shiftwidth=4
set expandtab
set cursorline
filetype on
filetype plugin indent on
```
